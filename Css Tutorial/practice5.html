Reason to disappear Bullets

When you set the display property of
  • elements to inline, the bullets disappear because the default list-item behavior is overridden. Bullets are associated with the display: list-item property, which is the default for
  • elements. When you change the display to inline, the element no longer behaves as a list item, and the bullets are removed.
